# Кодекс поведения

Проект следует [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Участники обязаны общаться уважительно, конструктивно воспринимать обратную связь
и не допускать преследования, дискриминации, угроз и публикации чужих персональных
данных. Недопустимый контент может быть удалён, а доступ нарушителя ограничен.

Для конфиденциального сообщения о нарушении используйте private vulnerability
reporting GitHub с пометкой «Code of Conduct» или запросите приватный контакт у
владельца репозитория через его GitHub-профиль. Все обращения рассматриваются
конфиденциально и добросовестно.

Этот документ адаптирует Contributor Covenant; FAQ доступен на
[contributor-covenant.org/faq](https://www.contributor-covenant.org/faq/).
