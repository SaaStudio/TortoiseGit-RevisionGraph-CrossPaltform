# Поддержка

- Ошибка или регрессия — используйте шаблон bug report.
- Предложение — используйте шаблон feature request.
- Вопрос по сборке — создайте discussion, если Discussions включены; иначе issue
  с минимальным воспроизводимым примером.
- Уязвимость — следуйте [SECURITY.md](SECURITY.md), не создавайте публичный issue.

Перед обращением проверьте существующие issues и повторите проблему на последней
версии основной ветки.
